####
MIXT
####

Write html components directly in python and you have a beautiful but controversial MIXTure

Based on ``pyxl``.

More text to come ;)


************
Installation
************

.. code-block:: shell

    pip install mixt


************
Developement
************

Clone the git project then:

.. code-block:: shell

    make dev

After having done some code:

.. code-block:: shell
    make tests

.. code-block:: shell

    make lint
